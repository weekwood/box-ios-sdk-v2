Overview

BoxFolderPickerViewController is a UI widget that allows quick and easy integration with Box. Displaying a BoxFolderPickerViewController provides a file browser and enables users to select a file or folder from their Box account.

The BoxFolderPickerViewController handles OAuth2 authentication by itself if you do not wish to authenticate users independently.

The BoxFolderPickerViewController makes extensive use of thumbnail support in the Box V2 API. Additionally, it can display assets from the BoxSDKResources bundle and if you wish to use the folder picker, you should include this bundle in your app.

Selection events are handled by the BoxFolderPickerDelegate delegate protocol.

authorizationViewController:shouldLoadReceivedOAuth2RedirectRequest:

Asks the delegate whether the view controller should load the OAuth2 redirect URL in the web view.

- (BOOL)authorizationViewController:(BoxAuthorizationViewController *)authorizationViewController shouldLoadReceivedOAuth2RedirectRequest:(NSURLRequest *)request

Parameters

authorizationViewController

The authorization view controller object managing the authorization process

request

The OAuth2 redirect URL request which the authorization web view is requesting to load

Return Value

YES if the redirect URL should be loaded, NO if it should not be.

Discussion

Asks the delegate whether the view controller should load the OAuth2 redirect URL in the web view.

If the delegate does not implement this method, the default is YES.

If you wish to support iOS 5, you should implement this method and handle the OAuth2 redirect URL in it. Web views on iOS 5 do not support loading custom URL schemes.

initWithSDK:rootFolderID:thumbnailsEnabled:cachedThumbnailsPath:fileSelectionEnabled:

Initializes a folderPicker according to the caching options provided as parameters. This folder picker is bound to one instance of the BoxSDK and thus, one BoxOAuth2Session.

- (id)initWithSDK:(BoxSDK *)sdk rootFolderID:(NSString *)rootFolderID thumbnailsEnabled:(BOOL)thumbnailsEnabled cachedThumbnailsPath:(NSString *)cachedThumbnailsPath fileSelectionEnabled:(BOOL)fileSelectionEnabled

Parameters

sdk

The SDK which the folder picker uses to perform API calls.

rootFolderID

The root folder where to start browsing.

thumbnailsEnabled

Enables/disables thumbnail management. If set to NO, only file icons will be displayed.

cachedThumbnailsPath

The absolute path where the user wants to store the cached thumbnails. If set to nil, the folder picker will not cache the thumbnails, only download them on the fly.

fileSelectionEnabled

Whether the user will be able to select a file or not while browsing his account. If not set to nil, the folder picker will cache the thumbnails at this path Not used if thumbnailsEnabled set to NO.

Return Value

A BoxFolderPickerViewController.

Discussion

Initializes a folderPicker according to the caching options provided as parameters. This folder picker is bound to one instance of the BoxSDK and thus, one BoxOAuth2Session.
